
Your-Search-Wrapped

A minimal web app that gives you a "Spotify Wrapped"-style summary for your AI tool usage. Currently, it supports ChatGPT data only.

## Features
- Upload your search data
- Get MBTI and topic analysis
- Simple, clean UI

## Getting Started

### Backend
1. Navigate to `BE/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`

### Frontend
1. Navigate to `FE/`
2. Install dependencies: `npm install`
3. Run: `npm run dev`

## Project Structure
- `BE/` - Backend API (Python)
- `FE/` - Frontend (React)
- `Model/` - Notebooks and models
